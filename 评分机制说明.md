# 多传感器融合评分机制 — 完整说明

## 一、执行周期

每 **5 秒** 计算一次（跟随遥测线程）。

## 二、衰减

每周期累积分 × **0.95**。连续 60 秒无新信号则清零。

## 三、信号与权重

| 信号 | 权重 | 触发条件 | 加分时机 |
|------|------|---------|---------|
| 烟雾传感器 | 60 | GPIO117 读到 0（报警） | 每次读到就加 |
| AI检测到fire | 30 | class_id=1, conf>0.30 | AI拍照的同一周期 |
| 温度上升速率 | 20 | 变化≥2°C 且 >3°C/分钟 | 满足条件就加 |
| 温度>33°C | 15 | 绝对温度>33°C | 每周期持续加 |
| 温度33~45°C | 10+（每多1°C+1） | 绝对温度在33~45°C之间 | 每周期持续加 |
| 温度基线偏高 | 2×(高出的度数) | 高于过去10次平均温度5°C以上 | 每周期持续加（上限20） |
| 湿度骤降 | 10 | 变化≥3% 且 >30%/分钟 | 满足条件就加 |
| 光照突变(夜间) | 10 | 夜间从暗变亮 | 单次事件 |
| 光照持续暗(白天) | 5 | 白天持续黑暗 | 每周期持续加 |
| 设备异常 | 10 | 风扇状态变化 + 烟雾同时报警 | 满足条件就加 |
| PIR有人(配合火灾信号) | 5 | PIR=1 + 烟雾/AI同时触发 | 同步场景加分 |
| PIR有人(夜间单独) | 2 | 夜间 PIR=1，无其他信号 | 每周期持续加 |

## 四、并发倍率

同一周期内 ≥2 个高权重信号（烟雾/AI/温度）同时触发 → 新信号总值 × **1.5**。

## 五、动态权重修正

| 时段 | 修正规则 |
|------|---------|
| 深夜 0:00~6:00 | 所有权重 ×1.5 |
| 白天 8:00~18:00 | PIR权重 ÷2 |
| 夏季 温度>30°C | 温度权重 ÷2 |

## 六、状态机

```
总分范围     → 状态    → 行为
──────────────────────────────────
0~29        → SAFE    → 正常监控，不做任何操作
30~59       → MONITOR → 关注，记录数据但不告警
≥60         → ALERT   → 发钉钉 + 开风扇 + 拍照上传
（ALERT 30秒后 → COOLDOWN → 再30秒后 → SAFE）
```

**进入 MONITOR 的快捷通道**（不依赖分数）：
- 烟雾传感器触发
- AI 任何检测结果 conf > 0.20

## 七、AI检测的特殊规则

| 规则 | 说明 |
|------|------|
| 检测间隔 | 每 **10 秒** 拍照一次上传腾讯云推理 |
| 加分时机 | 只在AI拍照的同一周期加分，中间空档不加 |
| 有效期 | 拍照后 **5 秒**内有效，超过不加分 |
| 阈值 | conf > **0.30** 才认可 |
| 置信度清除 | AI本次没检测到任何东西 → 立即清零 |
| 最佳帧 | 只保留置信度最高的帧（用于告警上传） |
| 滑动窗口 | 最近5次检测中 ≥2 次检测到 → 触发窗口通过 |

## 八、整体流程

```
                    AI拍照（每10秒）
                         │
            ┌────────────┴────────────┐
            │                         │
      检测到fire                 没检测到
       conf>0.30                清除置信度
       ↓+30分                       │
            │                         │
            └────────────┬────────────┘
                         │
              传感器读数（每5秒）
              温度/湿度/光照/PIR/烟雾
              各信号按规则加分
                         │
                   并发倍率×1.5
                   动态权重修正
                         │
                   衰减×0.95
                         │
                   总分≥60？
                   ┌─┴──┐
                  是    否
                   │     │
                ALERT  低于60→MONITOR
               发钉钉   或回到SAFE
               开风扇
               拍照
```
